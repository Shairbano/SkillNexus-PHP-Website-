# SkillNexus

SkillNexus is a PHP-based web application developed using **HTML, CSS, PHP, and phpMyAdmin**.  
The system is designed for companies that want to collaborate and form partnerships with other organizations, while also managing employees with specific expertise.

---

## Purpose of the System

The main purpose of SkillNexus is to:
- Connect companies through **affiliation requests**
- Allow individuals to **register as employees** based on their expertise
- Provide an **admin-controlled dashboard** to manage users, employees, and affiliations

---

## Main Features

The website consists of **two main user options**:

### 1. Register as Employee
- Users can register as employees after signing up and logging in
- The system asks for the employeeâ€™s **area of expertise**
- Employee data is stored and managed by the admin
- Employees can be searched based on expertise

### 2. Apply for Affiliation
- Companies can apply for affiliation with SkillNexus
- The applicant provides relevant company details
- The request remains **pending** until approved by the admin
- Once approved, the affiliation status is visible on the user dashboard

## ðŸ”¹ Admin Dashboard

After logging in as an **admin**, the following management options are available:

### Admin Functions:
- **Manage Users**
  - View all registered users
  - Add new users
  - Delete existing users

- **Manage Affiliation Requests**
  - View all affiliation requests
  - Accept or reject requests
  - Approved requests appear on the user dashboard

- **Manage Employees**
  - View registered employees
  - Search employees by expertise
  - Display relevant employees in a table format
